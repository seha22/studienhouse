## Information Architecture & UI Guidelines

### 1. Landing Page Structure
1. **Hero**
   - Headline: “Kuasai Matematika & Programming dalam 12 Minggu.”
   - Sub copy menegaskan jenjang SD–SMA + programming pemula.
   - Dual CTA: “Lihat Kurikulum” dan “Jadwalkan Konsultasi.”
   - Visual: mock dashboard dengan overlay stats (materi, guru, siswa).

2. **Keunggulan Utama**
   - Tiga kartu ringkas: Mode Hybrid, Mentor Tersertifikasi, Progress Tracker.
   - Ikon minimalis garis untuk tiap nilai jual.

3. **Jalur Pembelajaran**
   - Tab/accordion Matematika (SD, SMP, SMA) dan Programming (Pemula, Proyek Web).
   - Paparkan outcome tiap level + estimasi durasi.

4. **Persona Pengguna**
   - Tiga kartu: Admin, Guru, Siswa.
   - Jelaskan peran (CMS, unggah materi, monitor progres).

5. **Preview Dashboard**
   - Screenshot/illustration progress siswa dan module list.
   - Bullet feature: progress bar, jadwal kelas, notifikasi tugas.

6. **Testimonials & Stats**
   - Slider 2–3 siswa/guru.
   - Highlight angka lulus, rata-rata kenaikan nilai.

7. **FAQ**
   - Fokus pertanyaan mode online/offline, akses materi, integrasi MCP (admin CMS).

8. **Final CTA**
   - Form lead magnet (email + pilih jalur) + CTA “Download Silabus.”

### 2. Content Hierarchy & Grid
- Page width: 1200px container, 12-col grid dengan gutter 32px.
- Vertical rhythm: 80px antar section; 40px antar sub-block.
- Mobile stacking default; breakpoints di 768px & 1024px.

### 3. Tone & Messaging
- Gunakan bahasa langsung, menonjolkan outcome (naik nilai, projek web pertama).
- Sertakan kata kunci “Matematika SD/SMP/SMA”, “Programming Pemula”.
- Tampilkan CTA aktif di setiap section (misal “Pelajari Jalur SMP”).

### 4. Visual Direction
- Palet utama: Ivory background dengan aksen Orange (#F6B11C) & Mint (#CDEEC4).
- Secondary: Peach (#FEDCB3) untuk cards, Charcoal (#1C1A1E) untuk teks utama.
- Tipografi: Poppins (heading), Inter (body). Heading 64/48/28px.
- Ilustrasi: kombinasi foto siswa belajar + icon numerik/kode.
- Ikon linear 2px stroke untuk highlight (jadwal, laptop, progress).

### 5. Component Guidelines
- **Buttons**: pill radius 999px, primary orange, secondary outline charcoal.
- **Cards**: radius 32–44px, shadow halus, padding 32px.
- **Tabs**: rounded pill with active orange underline.
- **Progress bar**: gradient mint → orange, label persentase.
- **Testimonials**: avatar bundar, rating star, kutipan pendek.

### 6. Content Hooks per Section
- Hero stats: “120+ materi, 85 mentor, 24k siswa.”
- Jalur Matematika: highlight “Persiapan AKM & UTBK.”
- Programming: “Bangun web portofolio pertama.”
- Persona Admin: “Integrasi Supabase + MCP untuk CMS.”
- CTA akhir: “Tinggalkan email untuk jadwal kelas berikutnya.”

Dokumen ini menjadi dasar copywriting dan desain selanjutnya. Setelah final, lanjut ke definisi schema Supabase.
